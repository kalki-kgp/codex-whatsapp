How it works

// Everything included

Batteries included

No configuration gymnastics. Install once, text forever.

Voice notes → code

Whisper transcription turns WhatsApp voice messages into agent prompts. Hands-free coding.

30 saved sessions per chat

Name, save, and restore sessions. /resume auth-refactor brings back exactly where you left off.

Bot mode or self-chat mode

Dedicate a second WhatsApp number as your agent, or text your own number to yourself.

Daily memory rollover

Each day the agent compacts context into a carry-forward summary. Long projects, zero context loss.

In-chat upgrades

Agent pings you when a new version drops on PyPI. Reply /yes — it upgrades itself and restarts.

100% self-hosted

All data stays on your server. Credentials, sessions, memory — none of it touches a third party.

// Chat interface

Commands

Send these as WhatsApp messages from any allowed number.

whatsapp-agent — chat commands
> /new Archive current session and start fresh
> /compact Summarize context and continue — saves your context window
> /root /path/to/repo Switch the working directory for this chat
> /model <name> Change the model on the fly without clearing the session
> /resume List all saved sessions and restore one by name or id
> /search-session <q> Find and restore the most relevant saved session
> /memory Show this chat's long-term memory files and session ids
> /status Backend, model, root, active thread, session count
> /help Print all commands in-chat

// Get started

Up in 4 steps

Requires Linux + Python 3.10+ + Node 18+ + claude or codex already authenticated on your server.

1

Run the bootstrap

One command installs everything — uv, the CLI, and all runtime deps. Works on Debian/Ubuntu.

$ curl -fsSL https://raw.githubusercontent.com/kalki-kgp/whatsapp-agent-cli/main/scripts/bootstrap.sh | bash
2

Follow the interactive setup

Auto-detects Claude and Codex on your PATH. Pick bot vs self-chat mode, enter your allowed number, optionally enable Whisper for voice notes.

$ whatsapp-agent install # Arrow keys to navigate, Enter to confirm # Advanced options behind an opt-in toggle
3

Pair your WhatsApp account

A QR code prints in the terminal. Open WhatsApp → Linked Devices → scan. Credentials are saved and survive restarts.

$ whatsapp-agent pair # Scan the QR from WhatsApp → Linked Devices
4

Start texting

The service starts automatically. Check status anytime.

$ whatsapp-agent service status $ whatsapp-agent service logs # live tail $ whatsapp-agent doctor # diagnose any issues
